# Jogo Da Vida
Implementação do algoritmo do __Jogo da Vida de John Conway__ com __MPI__ para o segundo trabalho avaliativo da disciplina de Introdução a Processamento Paralelo e Distribuído na UFPel. O jogo da vida é um autómato celular desenvolvido pelo matemático britânico John Horton Conway em 1970.
